# Agent Configuration & Guidelines

When modifying or updating any agent skills, follow these rules:
1. Unless specifically instructed otherwise by the user, **always** modify the skills directly in the current working directory. Do NOT modify the global skills directory (e.g., `~/.agents/skills/`) or assume a specific folder name unless explicitly told to do so.
2. Before making modifications, verify the existence of the skill directory and `SKILL.md` file within the current directory context.
