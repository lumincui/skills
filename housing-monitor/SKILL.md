# Housing Market Monitor

## Overview

这个 skill 用于收集房地产市场数据、生成图表、分析趋势和政策影响。目标是做 **证据驱动的市场分析**，而不是凭感觉预测房价。

核心原则：

```text
先定城市和时间范围 → 找高置信数据源 → 交叉验证 → 可视化 → 分析趋势/政策/风险
```

## Phase 2: Data Sources

优先级：

| Priority | Source Type | Examples |
|---|---|---|
| A | 官方统计/政府平台 | 国家统计局、地方住建局、房地产信息平台 |
| B | 准官方/行业报告 | 中指研究院、贝壳研究院、乐有家研究中心 |
| C | 第三方聚合 | gotohui、安居客、链家公开页等 |
| D | 媒体报道 | 只作辅助，不作核心数据源 |

深圳常用来源：

| Data | Source | URL |
|---|---|---|
| 70 城房价指数 | 国家统计局 | `https://www.stats.gov.cn/sj/zxfb/` |
| 深圳成交/市场数据 | 深圳市住房和建设局 | `https://zjj.sz.gov.cn/xxgk/ztzl/pubdata/` |
| 房地产信息平台 | 深圳房地产信息平台 | `https://fdc.zjj.sz.gov.cn/` |
| 月度房价 | gotohui | `https://fangjia.gotohui.com/` |

数据必须记录：来源 URL、抓取日期、指标定义、单位、置信度。

## Phase 3: Metrics

核心指标：

| Metric | Unit | Notes |
|---|---|---|
| 二手房均价 | 元/㎡ | 注意挂牌价 vs 成交价 |
| 新房均价 | 元/㎡ | 注意网签/成交口径 |
| 二手房成交量 | 套/月 | 深圳荣枯线常用 5000 套/月参考 |
| 新房成交量 | 套/月 | 与供应节奏相关 |
| 租金 | 元/月 或 元/㎡/月 | 要说明户型/面积假设 |
| 租售比 | 房价 / 年租金 | 口径必须明确 |

租售比计算：

```text
房价总价 = 单价(元/㎡) × 面积(㎡)
年租金 = 月租金 × 12
租售比 = 房价总价 / 年租金
租金收益率 = 年租金 / 房价总价
```

注意：中文语境中“租售比”有时表达为 `1:N`，报告里要明确口径。

## Phase 4: Collection and Validation

步骤：

1. 搜索官方数据；
2. 下载/读取表格或网页；
3. 统一时间粒度（月/季度/年）；
4. 统一单位；
5. 与至少一个独立来源交叉验证；
6. 标注缺失、估算和口径差异。

搜索模板：

```text
<城市> <年份> 二手房 成交量 月度 官方
<城市> <年份> 新房 成交 均价 住建局
site:stats.gov.cn 70城 房价指数 <城市>
site:zjj.sz.gov.cn 深圳 二手房 成交 月度
```

## Phase 5: Visualization

优先使用已有脚本：

```bash
python3 scripts/generate_charts.py
```

或按任务生成新图表。图表要求：

- 标题包含城市和时间范围；
- 坐标轴标单位；
- 标注关键政策节点；
- 数据来源写在图注或报告中；
- 不同口径数据不要画在同一轴上误导比较。

建议图表：

1. 二手房均价趋势；
2. 新房均价趋势；
3. 二手房/新房成交量对比；
4. 租金趋势；
5. 租售比或租金收益率；
6. 政策节点 vs 成交量/价格。

## Phase 6: Analysis Framework

### Price and Volume

| Pattern | Interpretation |
|---|---|
| 量价齐升 | 需求回暖，卖方议价增强 |
| 量升价平 | 底部活跃，可能筑底 |
| 量跌价跌 | 买方观望，市场下行 |
| 量平价涨 | 可能由结构性成交或供给变化造成 |

### Policy Impact

分析政策时分三层：

1. 政策内容：限购、限贷、税费、利率、首付；
2. 作用机制：降低门槛、改善预期、释放需求；
3. 数据响应：成交量先动，价格后动，租金更慢。

深圳重点节点：

- 2021-02 二手房参考价制度；
- 2022-11 金融 16 条；
- 2023-09 认房不认贷；
- 2024-09 930 政策。

### Forecasting

预测必须给置信度和风险，不要单点断言。

```text
短期 1-3 月：看政策后成交量和带看热度
中期 6-12 月：看库存、利率、收入预期
长期 1-3 年：看人口、产业、供地和政策方向
```

## Common Pitfalls

1. **挂牌价当成交价**：必须区分口径。
2. **第三方数据不标置信度**：容易误导。
3. **只看价格不看成交量**：量往往先于价变化。
4. **预测没有风险提示**：房地产受政策和信贷影响很大。
5. **租售比口径混乱**：要说明 `房价/年租金` 还是收益率。
6. **图表无单位/来源**：不可交付。
